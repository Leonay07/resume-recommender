## 项目总览（Resume Recommender）

### 项目目标
本项目旨在开发一个能够根据用户上传的简历自动推荐岗位的网页系统。  
系统部署于 Hugging Face Spaces，具备端到端功能演示能力。

### 系统组成
| 模块 | 内容 |
|------|------|
| 前端 | 使用 React + Vite 构建用户界面 |
| 后端 | 使用 FastAPI 提供 API 接口与 NLP 模型调用 |
| 模型 | NLP 推荐逻辑（由团队成员负责实现） |
| 部署 | 使用 Docker 构建并运行于 Hugging Face Spaces |

### 模块分工
1. 项目初始化  
2. 前端页面结构与组件  
3. 后端接口骨架  
4. 前后端联调  
5. 模型接口预留  
6. 部署规范  

### 文件结构总览
```
resume-recommender/
│
├── frontend/      # 前端项目
├── backend/       # 后端项目
├── docs/          # 技术文档与规范
└── Dockerfile     # 部署文件
```

### 模块依赖关系
| 依赖顺序 | 说明 |
|-----------|------|
| 01 → 02 | 需先建立项目结构再搭建前端 |
| 02 → 04 | 前端完成后可进行联调 |
| 03 → 04 | 后端接口完成后可联调 |
| 04 → 05 | 联调完成后再接入 NLP 模型 |
| 05 → 06 | 模型接口完成后可统一打包部署 |

### 评分重点（课堂项目）
- 系统可运行（deployable）  
- 前后端逻辑完整（end-to-end）  
- 工程可扩展（scalable）  
- 成本与稳定性控制（cost-efficient & reliable）
